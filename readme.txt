Project Name: Marathi News clustering.
March 2016 - May 2016
Description:
	1) Clustering of news from "Marathi" language( A native language of Maharashtra, India ) 
	2) K-means Model is trained and used to classify the news from following categories:
		a) Sport
		b) Political
		c) Entertainment
		d) economical

Overview:
	1) File 'stopword.txt' contains stop words used in Marathi language.
	2) Folder 'Arff file' contains data to train and test model.
	3) Folder 'FinalProject' contains actual source code.
	4) Required jar weka.jar can be downloaded from http://prdownloads.sourceforge.net/weka/weka-3-8-2.zip.
